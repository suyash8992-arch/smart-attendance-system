# Code of Conduct

## Our Pledge

We are committed to fostering a welcoming, respectful, and collaborative environment for everyone who contributes to the **Smart Attendance System** project.

Regardless of experience level, background, or identity, all contributors are expected to participate in a professional and respectful manner.

---

# Our Standards

Examples of behavior that contribute to a positive environment include:

- 🤝 Treat everyone with respect and professionalism.
- 💬 Be open to constructive feedback and discussion.
- 📖 Share knowledge and help others learn.
- 🛠 Focus on improving the project through collaboration.
- 🌟 Encourage teamwork and inclusive participation.

Examples of unacceptable behavior include:

- Harassment, discrimination, or personal attacks.
- Offensive or abusive language.
- Trolling or intentionally disruptive behavior.
- Sharing private or confidential information without permission.
- Plagiarism or submitting work that you do not have the right to share.

---

# Project Expectations

When contributing to this repository, please:

- Follow the project's coding standards.
- Test your changes before submitting them.
- Keep documentation up to date.
- Use clear and descriptive commit messages.
- Respect the decisions made through project discussions.

---

# Reporting Issues

If you encounter inappropriate behavior or have concerns regarding this Code of Conduct, please open a private discussion with the project maintainers or contact the repository owner.

All reports will be handled respectfully and confidentially whenever possible.

---

# Scope

This Code of Conduct applies to:

- GitHub Issues
- Pull Requests
- Project Discussions
- Documentation
- Any other interactions related to this repository

---

# Enforcement

Project maintainers may remove comments, commits, or contributions that violate this Code of Conduct and may restrict participation when necessary to maintain a positive and productive environment.

---

# Acknowledgements

This Code of Conduct is inspired by the principles of the Contributor Covenant while being adapted for the Smart Attendance System project.

---

# Thank You

Thank you for helping us build a respectful, collaborative, and professional open-source project.

Every positive contribution—whether code, documentation, testing, or feedback—is appreciated.
